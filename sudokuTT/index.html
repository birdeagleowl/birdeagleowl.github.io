sudokuTT classic SUDOKU puzzle

Challenge your brain with our Sudoku game sudokuTT! A fun and challenging puzzle game that will keep you entertained.


1. Press the play button to start the game.
2. Select a blank space and enter the numbers.


3. After entering all the numbers, confirm the answer with the check button.
4. If an incorrect message appears, you can either restart the game or continue playing.


5. When the correct message appears, you can start a new game.
6. Start a new game.

7. This is the settings screen.
